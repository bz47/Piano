HTML FORM #1

First practice making a form. 5/31/2023

    REQUIREMENTS:
  • action = tells where to send info & Method = tells what to do with info
  • a input form can be created with input and brackets. input forms need a type attribute and a name attribute
  • a label element to define what the input is for. Labels have opening and closing brackets and need a for attribute with a value of the inputs's id.
  • input can have type of password/text/number/range/radio/checkbox...







What is the sum of 1 + 1 ?









    TIPS and OPTIONS:
  • the name attribute is only used to send the info
  • range inputs can have a min and max and step
  • checkboxes for can have same name but ids must be different. also values must be set but wont be seen unless declared in label
  • dropdown lists use select and option tags
  • datalists are convenient bc users dont have to scroll through options, they can type their input
  • datalist inputs take a list attribute whose values is passed to the datalist tag
  • number inputs can use min and max to set parameters
  • text inputs can use minlength and maxlength to set parameters
  • use pattern attribute plus regEx to check if input fits required parameters
